# ujkweb
Aplikasi sistem manajemen data produk berbasis web 
Nama : Ina Nasfa Ajizah
Nim : 220103127
Kelas : TI22A4
Sistem ini digunakan untuk mengelola data produk secara terstruktur, meliputi proses pencatatan produk baru, penyimpanan informasi produk seperti nama, harga, dan stok, pembaruan data produk, hingga penghapusan data yang sudah tidak diperlukan. Dengan adanya sistem ini, pengelolaan data produk menjadi lebih efisien, akurat, dan aman karena hanya dapat diakses oleh pengguna yang telah melakukan login.
